# LLM Wiki

This wiki is for future agents working on `auto-toss`.

Use `docs/SOT` for stable source-of-truth documents. Use this wiki for daily work notes, implementation details, class/method notes, architecture notes, infra notes, and recorded dead ends.

## Folder Map

- `work-units/`: daily task notes describing what changed and why.
- `dead-ends/`: debugging notes, false starts, caveats, and discarded approaches.
- `classes/`: class and method level notes.
- `architecture/`: subsystem architecture notes.
- `infra/`: storage, runtime state, local environment, and operational notes.

## Rule

Documentation updates are mandatory after implementation work. If behavior changes and this wiki or SOT remains stale, the task is incomplete.
